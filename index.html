How to Install

On Steam Deck

Requirements

  • Decky Loader
  • CSS Loader
  • Audio Loader

To customize your Steam Deck's theme, you will first need Decky Loader. Decky Loader is a plugin utility. Decky plugins expand the functionalities of SteamOS' Gaming Mode. Installation instructions for Decky are linked on their official site above.



Once you have Decky installed, you will need the CSS Loader and Audio Loader plugins from the Decky plugin shop. These will allow you to install custom themes and sound effects respectively to transform your SteamOS Gaming Mode experience.





Download or update both the CSS Theme and SFX Pack very quickly by clicking the button above to copy the "1-Click Command".

Then open the Konsole app, right-click to Paste, and hit Enter. Note that this will overwrite the previous version of both the CSS Theme and Audio Pack.



ALTERNATIVELY, follow the steps below to download and setup both manually:


Download the "MHOnDeck CSS Theme" and unzip the contents onto home/deck/homebrew/themes
Download the "MHOnDeck SFX Pack" and unzip the folder contents onto home/deck/homebrew/sounds


P.S. If you do not see either folder, create them manually following the directory structure above.



Next, return to Gaming Mode, access the CSS Loader plugin from the Decky store, then "Refresh" at the bottom. You will now see "MHOnDeck CSS Theme" as an option which you can enable.

P.S. CSS Loader allows multiple "Profiles" so you can toggle between different sets of CSS themes, to keep your SteamOS Gaming Mode experience fresh at any time. Also, the "MHOnDeck CSS Theme" is designed to be used FULLY standalone. If you have other CSS Loader themes enabled at the same time, certain elements of this theme MAY break.

Lastly, on Decky, visit the Audio Loader plugin and click on "Manage Packs", then "Refresh". Check the dropdown again for "MHOnDeck SFX Pack" and select it.
